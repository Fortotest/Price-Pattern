# **App Name**: PricePattern Studio

## Core Features:

- Template Pattern Generation: Generate mathematically predefined candlestick patterns (SNR Waves, Basic & Complex structures) from a library of templates, directly populating the chart canvas.
- AI Pattern Augmentation Tool: A generative AI tool that, given a chosen pattern template or a text description of a market scenario, suggests or subtly adjusts candle parameters (body, wick, offset Y) to create more nuanced and realistic variations for price action replay and visualization.
- Smart Manual Candlestick Editor: Control individual candle properties (Type: Bullish/Bearish/Doji, Body, Wick Top/Bottom, Offset Y) with an 'Auto-Snap' function that sets new candles' open price to the previous close.
- 4K Live Chart Renderer: Dynamically render high-resolution 3840 x 2160 pixel (4K UHD) candlestick charts on an HTML5 canvas, featuring precise spacing, auto-centering, and dynamic zoom via CSS 'object-fit: contain'.
- Vector SVG Export: Export the currently displayed chart as a pure `<svg>` code file, ensuring a 100% transparent background, tiny file size, and infinite resolution for any display or print use.
- Cinematic Video Replay & Export: Record and export tick-by-tick animated price action of chart formation to a `.webm` video using MediaRecorder API, with customizable speed control (0.5 to 2.5 seconds per candle) at 60 FPS and 30 Mbps bitrate.

## Style Guidelines:

- A sophisticated light color scheme that emphasizes clarity and professionalism, drawing from the structured nature of financial data and precise analytics.
- Primary interactive elements will use a composed blue (#5590C0) to convey reliability and data focus.
- The background will be a very light, almost white blue (#EBF1F4), creating a clean canvas that promotes readability without stark contrast.
- An accent color in a calm cyan-green shade (#61D4BD) will highlight key actions and information, offering a fresh, clear counterpoint to the primary blue.
- The primary font for all text, including headlines and body, will be 'Inter', a grotesque-style sans-serif known for its modern, objective, and highly legible appearance on digital screens, perfect for conveying precise financial data.
